論理回路シュミレーター(ゴミ)の操作説明

ピンクの箱が4つありますが、左から"AND","OR","NAND","NOR"となっています。

--回路を選択する--
ピンクの箱を１回クリックすると、赤くなります。この状態で上のいずれかの回路の箱をクリックすると、赤くなった箱に対応した論理回路が箱に表示されます。

選択していない状態（すべての箱がピンクのとき）で回路の箱をクリックすると箱の中身をNULLにすることができます。

また、回路に接続されている円は電球を意識しており、下記で説明する入力を実行した際の出力結果を視覚的にわかるようにするためのものです。

--回路を実行する--

ピンクの箱の下にあるボタンで0(FALSE),1(TRUE)を選択して入力します。
"1-上"は左上の回路の上の線に入力する値
"1-下"は左上の回路の下の線に入力する値
"2-上"は左下の回路の上の線に入力する値
"2-下"は左下の回路の下の線に入力する値
となっています。
それらの入力が終わったら、"ここを押すと上の入力で動かすよ！"のボタンを押していただくと、上記の入力で実行します。
また、回路が選択されていない場合は"0(FALSE)"が出力されるようになっています。

実行結果は回路内の円の色｛黄色(TRUE),黒(FALSE)｝で確認することができます。
